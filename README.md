# Volume Horror UX

A gallery of the world's worst volume control interfaces, inspired by [The Worst Volume Control UI in the World](https://uxdesign.cc/the-worst-volume-control-ui-in-the-world-60713dc86950).

**[Live Demo](https://zebiv.com/volume-horror/)**

## Controls

Each widget is a fully interactive volume control that actually adjusts playback volume. Includes:
radio buttons, math equations, dice rolling, rapid tapping, coloring grids, Snake game, projectile launching, and more.

## Tech

- Vanilla HTML/CSS/JavaScript, Canvas 2D, Web Audio API
- ES modules, no build step
